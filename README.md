
# ## **Food Delivery System (C++/Cpp)**

 **Project Overview**

A feature-rich console-based Food Delivery Management System developed in C++ using modular programming, file handling, and real-world business logic.  
The system simulates a complete online food ordering platform with customer and admin functionalities.


# ## **Key Features**

# ## **Customer Features**
- View Food Menu
- Search Food Items
- Place Orders
- Wallet Recharge System
- Promo Code Discounts
- Order Tracking
- Order Cancellation
- Customer Feedback & Ratings

---

# ## **Admin Features**
- View All Orders
- Update Order Status
- Manage Food Availability
- View Delivery Riders
- Sales Summary & Analytics
- Generate Business Reports
- View Customer Feedback

---

# ## **Payment & Delivery Features**
- Cash on Delivery
- Wallet Payment System
- Promo Code Validation
- Automatic Discount Calculation
- GST/Tax Calculation
- Automatic Rider Assignment
- Estimated Delivery Time
